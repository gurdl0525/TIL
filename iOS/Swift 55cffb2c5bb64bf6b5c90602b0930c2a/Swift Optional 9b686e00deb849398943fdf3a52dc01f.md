# Swift Optional

---

<aside>
🧐 **옵셔널이란?**

---

**이 변수에는 값이 들어갈 수도 있고, 아닐 수도 있어(nil)라는 뜻**

- nil을 표현하기 위한 수단 : ?
    
    ```swift
    let someData: Int?
    ```
    
</aside>

```swift
var optionalString: String = nil //error
var optionalString: String? = nil //not error
```
